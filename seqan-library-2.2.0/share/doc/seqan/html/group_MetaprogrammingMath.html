uint64_t Log2<NUMERUS>::VALUE;

Compute ceiled logarithm to base 2 using metaprogramming.

Template Parameters

NUMERUS int64_t value to use for the numerus.

Returns

uint64_t ceil(log2(NUMERUS))

Example

    std::cout << Log2<10>::VALUE << "\n";  // => "4"
