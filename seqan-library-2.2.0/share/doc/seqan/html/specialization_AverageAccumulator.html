Detailed Description

The average of an empty sequence is defined to be 0.

Examples

This program shows how to use the Average Accumulator.

Accumulator<int, AccuAverage> acc;
push(acc, 1);
push(acc, 2);
push(acc, 3);
std::cout << "average: " << average(acc) << "\n"
          << "sum:     " << sum(acc) << "\n"
          << "count:   " << count(acc) << "\n";

The output is then:

average: 2
sum:     6
count:   3
